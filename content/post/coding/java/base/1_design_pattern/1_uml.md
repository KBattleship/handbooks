# 关于UML图
学习设计模式前，UML还是需要学习了解一下。
### UML
`UML(Unified Modeling Language),即统一建模语言。是让系统可视化、让规格与设计文档化的变现方法。`
#### 旨在目的：
1. 为用户提供现成的、有表现力的可视化建模语言，以便开发与交换有意义的模型。
2. 为核心概念提供可扩展性与特殊化机制。
3. 独立特定的编程语言与开发过程。
4. 鼓励面向对象工具市场的发展。
5. 支持更高层次的开发概念。
6. 为了解建模语言提供一个正式环境。
7. 整合最佳的工作方法。
