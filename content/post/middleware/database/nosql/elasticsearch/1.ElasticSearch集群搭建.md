# ElasticSearch集群搭建
### 注：
```shell
#A:修改/etc/security/limits.conf
#<domain>      <type>  <item>         <value>
   *            soft    nofile          65536
   *            hard    nofile          131072
   *            soft    nproc           2048
   *            hard    nproc           4096

#B:修改/etc/sysctl.conf
vm.max_map_count=262144
# 保存执行：
sysctl -p
# 或者
sysctl -w vm.max_map_count=262144

```
